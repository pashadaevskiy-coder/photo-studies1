# Тестовые изображения для сайта фотостудии "PHOTO STUDIES"

## Описание

Эта папка содержит тестовые изображения для демонстрации функциональности сайта:

### Главная страница (Hero)
- `hero.jpg` - фоновое изображение для секции Hero

### Портфолио
- `portfolio/1.jpg` - индивидуальная съёмка
- `portfolio/2.jpg` - love story
- `portfolio/3.jpg` - семейная съёмка
- `portfolio/4.jpg` - портретная съёмка
- `portfolio/5.jpg` - событийная съёмка
- `portfolio/6.jpg` - беременная фотосессия
- `portfolio/7.jpg` - выпускной
- `portfolio/8.jpg` - другие работы

### Другие изображения
- `services/1.jpg` - мини пакет
- `services/2.jpg` - стандарт пакет
- `services/3.jpg` - премиум пакет

## Как использовать

1. Замените тестовые изображения на реальные фотографии
2. Оптимизируйте изображения для веба (сжатие, WebP формат)
3. Обновите пути в HTML шаблонах
4. Настройте lazy loading для лучшей производительности

## Размеры изображений

- **Hero**: 1920x1080px (fullscreen)
- **Портфолио**: 800x600px (aspect ratio 4:3)
- **Карточки услуг**: 600x400px

## Оптимизация

Для лучшей производительности рекомендуется:
- Использовать WebP формат
- Сжимать изображения (сайт должен загружаться < 3 секунд)
- Добавить alt текст для SEO
- Настроить lazy loading
